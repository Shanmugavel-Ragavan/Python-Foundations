# Python-Foundations

A comprehensive collection of Python scripts designed to take a learner from basic syntax to advanced Object-Oriented Programming (OOP) concepts. This repository serves as a structured guide for mastering the fundamentals of the Python programming language.

 📚 Course Content

The repository is organized numerically to provide a step-by-step learning path:

1. Python Basics
- Fundamentals: Keywords, Input Statements, Comments, and Type Casting.
- Strings: String functions and manipulation.
- Operators: 
  - Arithmetic, Assignment, and Comparison/Relational operators.
  - Logical, Bitwise, Identity, and Membership operators.

2. Control Flow & Iteration
- Conditional Statements: `if`, `if-else`, `elif`, and Nested If statements.
- Loops: `while` loops, `for` loops, and Nested For loops.
- Loop Control: `break`, `continue`, and the `range()` function.
- Advanced Loops: `while-else` and `for-else` constructs.

3. Data Structures
- Collections: Detailed implementations of `List`, `Tuple`, `Set`, and `Dictionary`.

4. Functions & Modules
- Core Concepts: Creating and using functions.
- Standard Libraries: Working with `date_time` and `math` functions.
- Error Handling: implementing `try-except` blocks for robust code.

5. Object-Oriented Programming (OOP)
- Basics: Classes, Objects, Class Attributes, and Instance Attributes.
- Methods: 
  - `__init__` method, Instance methods, Class methods, and Static methods.
  - Use of `@property`, `@classmethod`, and `@staticmethod` decorators.
- Advanced OOP:
  - Abstraction and Encapsulation.
  - Inheritance: Single, Multiple, and Multilevel inheritance.
  - Function Overriding and handling the Diamond Problem.
  - Operator Overloading and Abstract Base Classes (ABC).

6. File Handling
- Reading, writing, and deleting files in Python.

 🚀 Getting Started

Prerequisites
- Python 3.x installed on your system.

How to use
1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```
2. Navigate to the directory:
   ```bash
   cd Python-Foundations
   ```
3. Run any script to see the implementation:
   ```bash
   python 1_keywords.py
   ```

 🛠️ Project Structure
The files are prefixed with numbers (e.g., `1_keywords.py`, `52_file.py`) to indicate the recommended order of study.
